
**Installation**

Download and install NodeJS
     
     https://nodejs.org/en/download/

Check nodejs version

    node -v 
    v6.10.0

Check npm version
 
    npm -v
    4.0.5
    
Install latest angular-cli globally
     
    npm i -g @angular/cli@1.0.0-rc.0

Check ng cli version
    
    ng -v
